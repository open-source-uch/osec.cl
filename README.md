# OSEC website

## Hecho con:

- [Astro](https://astro.build/)
- [TailwindCSS](https://tailwindcss.com/)
- [Iconify Design](https://iconify.design/)
- [pnpm](https://pnpm.io/)

# Ejecutar el entorno dev:

Primero es necesario instalar pnpm

```bash
pnpm install
pnpm run dev
```
